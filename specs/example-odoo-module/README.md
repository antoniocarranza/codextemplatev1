# Example Odoo Integration

## Objetivo

Servicio Python para integrarse con Odoo v16 mediante XML-RPC.

## Requisitos

- configuración por variables de entorno
- cliente reutilizable XML-RPC
- separación entre capa de cliente, servicios y casos de uso
- pruebas unitarias para piezas críticas

## Entregables

- cliente base
- ejemplo de autenticación
- ejemplo de lectura y escritura
